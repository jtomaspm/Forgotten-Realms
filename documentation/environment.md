# Environment 

## Environment Variables

Create a `.env` file in the root directory of the project (next to `infrastructure/docker-compose.yml`) with the following content:

```ini
MYSQL_ROOT_PASSWORD=...
MYSQL_DATABASE=...
MYSQL_USER=...
MYSQL_PASSWORD=...
MYSQL_HOST=...

MYSQL_PORT=...
BACKEND_HTTP_PORT=...

GAME_ADMIN_NAME=...
GAME_ADMIN_EMAIL=...

GITHUB_CLIENT_ID=...
GITHUB_CLIENT_SECRET=...
```
